연산자 정리

산술 연산자

  • + : 더하기
  • - : 빼기
  • * : 곱하기
  • / : 나누기
  • % : 나누고 나머지 구하기
  • ++ : 피연산자를 1 증가시킵니다.
  • -- : 피연산자를 1 감소시킵니다.

콘솔창에 입력해 보새욘 ㅋㅋ

* 나누기 연산자와 나머지 연산자 비교하기
var numberOne = 15 / 2;
var numberTwo = 15 % 2;

* 변수 뒤에 증가 연산자 붙이기
var a = 10
var b = a++ + 5

* 변수 앞에 증가 연산자 붙이기
var c = 10
var d = ++c + 5 // d는 16, c는 11

비교 연산자

피연산자 2개의 값을 비교해서 참이나 거짓으로 결괏값을 반환

  • == : 피연산자가 서로 같으면 true
  • === : 피연산자도 같고 자료형도 같으면 true
  • != : 피연산자가 서로 같지 않으면 true
  • !== : 피연산자가 같지 않거나 자료형이 같지 않으면 true
  • 그 외 <, <=, > , >=

할당 연산자

  • = : 연산자의 오른쪽 값을 왼쪽 변수에 할당

    • 할당연산자와 산술연산자 합쳐서 표현하기
    단, 저장하려는 변수와 계산식 변수가 같을 때만 사용 가능

  • +=, -=, *=, /=, %=

연결 연산자

기호 : +

둘 이상의 문자열을 합쳐서 하나의 문자열로 만드는 연산자

ex) birthYear + "년에 태어난 사람의 나이는" + age + "세 입니다."

논리 연산자

불리언(boolean)연산자라고도 하며 true, false를 처리하는 연산자

  • OR연산자 - 기호 || : 피연산자중 하나만 true여도 true
  • AND연산자 - 기호 && : 피연산자가 모두 true일경우에만 true
  • NOT연산자 - 기호 ! : 피연산자의 반댓값을 지정
