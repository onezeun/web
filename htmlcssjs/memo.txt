parseInt() : 괄호 안에 값을 정수로 변환하는 함수

for문 : 초깃값 → 조건 → 명령 → 증가식 순서로 진행
for(초깃값; 조건; 증가식) {
  실행할 명령
}

while문 : 조건이 true인 동안 명령을 반복, 조건이 false라면 명령은 한 번도 실행하지 않을 수 있음
while(조건) {
  실행할 명령
}

do~while문 : 조건이 맨 뒤에 붙음 do문은 명령을 한 번 실행한 후 while문에서 조건을 체크
그러므로 조건이 false라 하더라도 일단 명령은 최소한 한 번 실행됨
do {
  실행할 명령
} while(조건)
