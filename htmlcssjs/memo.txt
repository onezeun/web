for문 : 초깃값 → 조건 → 명령 → 증가식 순서로 진행
for(초깃값; 조건; 증가식) {
  실행할 명령
}

while문 : 조건이 true인 동안 명령을 반복, 조건이 false라면 명령은 한 번도 실행하지 않을 수 있음
while(조건) {
  실행할 명령
}

do~while문 : 조건이 맨 뒤에 붙음 do문은 명령을 한 번 실행한 후 while문에서 조건을 체크
그러므로 조건이 false라 하더라도 일단 명령은 최소한 한 번 실행됨
do {
  실행할 명령
} while(조건)

여러 동작을 묶는 덩어리, 함수
함수를 사용하면 각 명령의 시작과 끝을 명확하게 구별할 수 있고, 묶은 기능에 이름을 붙여서 어디서든 같은 이름으로 명령을 실행할 수 있다.

변수의 적용 범위
스코프 (scope) : 변수를 선언하고 사용할 때 변수가 적용되는 범위
지역변수 (local variable) : 로컬 변수라고도 하며 한 함수 안에서만 사용할 수 있는 변수 (var와 함께 변수 이름 지정)
전역변수 (global variable) : 글로벌 변수라고도 하며 스크립트 소스 전체에서 사용할 수 있는 변수

let : 블록({}) 안에서만 쓸 수 있는 변수 재할당은 가능하지만 재선언이 안됨, 호이스팅이 없음

변수 사용
- 전역 변수는 최소한으로 사용
- var 변수는 함수의 시작 부분에서 선언
- for문에서 카운터 변수는 var보다 let 변수로 선언
- ES6를 사용한다면 var보다 let를 사용하는 것이 좋다.

즉시실행함수 : 한 번만 실행하는 함수일때 함수를 정의하면서 동시에 실행하는거 ~~

형식

(function() {
  명령
}());

또는

(function(매개변수) {
  명령
}(인수));

화살표 함수
=>표기법을 사용한 함수 선언. 익명 함수에서만 사용 가능
(매개변수) => {함수내용}
